Select Menus

MDC Select provides Material Design single-option select menus.

Import

Design & API Documentation

mdc-select

Properties Description
id: string Unique Id of the select (auto generated if not supplied).
name: string Name of the select.
placeholder: string Text shown if no value has been selected.
value: string Value of the checkbox.
autosize: boolean If true, the width of the select and it's menu are calculated and set appropriately. (Default is true)
tabIndex: number Set the underlying tab index of the select. (Default is 0)
floatingLabel: boolean Whether or not to show or hide a floating placeholder.
box: boolean Styles the select as a box select.
Methods Parameters
focus() Sets focus to the select component.
getValue(): string Returns the value of the currently selected option, or an empty string if no option is selected.
getSelectedIndex(): number Returns the index of the currently selected option. Returns -1 if no option is currently selected.
isDisabled(): boolean Returns whether or not the select is disabled.
setDisabled(disabled: boolean) Enables/disables the select.
setFloatingLabel(floatingLabel: boolean) Enables/disables the floating label.
setPlaceholder(text: string) Sets the placeholder text.
setBox(box: boolean) Styles the select as a box select.
setValue(value) Sets the selected item by value.
Events Description
change(index: number, value: string) Event emitted on any value change.
selectionChange(index: number, value: string) Event emitted if user changed the value.
